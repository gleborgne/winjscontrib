page actions and page links

Page actions and page links are shortcuts to help you bind some user interactions directly into your markup by adding some attributes.

When you are using our custom navigator, those attributes are resolved during page construction, but you can trigger them using WinJSContrib.UI.bindActions.

The following button is bound to the "alertMe" function on the page. You could optionnaly add arguments to the action using strict JSON.

If you need to provide argument to a page action, you could also use a function.
